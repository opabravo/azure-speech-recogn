# 說明

簡單的語音轉文字CLI工具，使用AZURE語音服務

### 已包裝程式下載

 [按此下載](https://github.com/opabravo/azure-speech-recogn/releases/download/1.1/vtt.exe)

# 開發環境

 * python 3.10

`pip install -r requirements.txt`

# Compile
 * 引入Azure語音服務套件所需的DLL
 
```pyinstaller main.spec```
